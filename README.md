## Argentina Housing Dashboard

Interactive data analytics dashboard for real estate market analysis in Buenos Aires, Argentina.

## Project Overview

This project scrapes, analyzes, and visualizes housing data from Mercado Libre to provide insights into the Buenos Aires real estate market. It includes automated data collection, interactive visualizations, and machine learning-based price predictions.

## Work in Progress

This project is currently under active development. Check back soon for updates!

## Features (Planned)

- Automated daily/weekly scraping of property listings
- Interactive dashboard with filters and visualizations
- Price prediction model
- Geographic heatmaps
- Trend analysis over time

## Tech Stack

- **Data Collection**: Python, BeautifulSoup/Selenium
- **Database**: SQLite
- **Dashboard**: Streamlit
- **Automation**: GitHub Actions
- **Deployment**: Render
